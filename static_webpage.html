Welcome to My Styled Webpage

This simple webpage is an example of Web 1.0 design, which focuses on static content and basic HTML structures. Back in the early days of the web, pages like this were the norm, offering straightforward information without any interactivity.

Web 1.0 sites were often used to share knowledge, create personal portfolios, and serve as directories. They featured static text, simple images, and hyperlinks to other resources.
